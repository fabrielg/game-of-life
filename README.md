## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements

```bash
python3 -m pip install --upgrade pip
```

Then:
```bash
pip install -r requirements.txt
```

## Usage

```python
python3 main.py
```
